## 1. Objective

The objective of the Nexora website is to:

* **Establish premium brand positioning** as a high-end web development agency
* **Showcase technical expertise** through sophisticated design and smooth interactions
* **Convert high-value clients** through compelling portfolio and clear CTAs
* **Differentiate from competitors** with a distinctive "Dark Mode Premium" aesthetic
* **Build trust** through professional presentation and proven work examples

## 5. Design System Specifications

### 5.1 Color Palette

| Color | Hex Value | Usage |
|-------|-----------|-------|
| Primary Background | #000000 | Main background, deep black |
| Secondary Background | #0a0a0a | Cards, sections (subtle contrast) |
| Accent Cyan | #00D4FF | CTAs, highlights, typewriter text |
| Accent Glow | rgba(0, 212, 255, 0.3) | Button shadows, border glows |
| Text Primary | #FFFFFF | Headings, important text |
| Text Secondary | #A0A0A0 | Body text, descriptions |
| Glass Background | rgba(255, 255, 255, 0.05) | Cards, navbar |
| Glass Border | rgba(255, 255, 255, 0.1) | Subtle borders |

### 5.2 Typography

| Element | Specification |
|---------|--------------|
| Font Family | Inter, Geist, or similar geometric sans-serif |
| Hero Heading | 64-72px, Extra Bold (800 weight) |
| Section Headings | 36-48px, Bold (700 weight) |
| Subheadings | 18-24px, ALL-CAPS, Letter-spacing 0.2em |
| Body Text | 16-18px, Regular (400 weight), #A0A0A0 |
| Buttons | 14-16px, Medium (500 weight) |
| Labels | 12-14px, CAPS, Letter-spacing 0.15em |

**Hierarchy Example:**
```
[LABEL]       WEB DEVELOPMENT & DIGITAL SOLUTIONS    (12px, caps, #A0A0A0)
[SUB-LABEL]   A COMPLETE                             (18px, caps, letter-spaced)
[HEADING]     Modern Web Solutions for               (64px, bold, #FFFFFF)
[HIGHLIGHT]   BUSINESS                               (64px, bold, #00D4FF, animated)
[BODY]        Clean, scalable, performance-focused...(16px, regular, #A0A0A0)
```

### 5.3 Visual Effects

#### Glassmorphism
```css
.glass {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 12px;
}
```

#### Gradient Buttons
```css
.btn-primary {
  background: linear-gradient(135deg, #00D4FF, #0099FF);
  box-shadow: 0 0 30px rgba(0, 212, 255, 0.4);
  border-radius: 50px;
  padding: 14px 32px;
  transition: all 0.3s ease;
}

.btn-primary:hover {
  box-shadow: 0 0 50px rgba(0, 212, 255, 0.6);
  transform: translateY(-2px);
}
```

#### Edge Glow Effect
```css
body::before {
  content: '';
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: 200px;
  background: radial-gradient(ellipse at top, rgba(0, 212, 255, 0.15), transparent);
  pointer-events: none;
  z-index: -1;
}

body::after {
  content: '';
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
  height: 200px;
  background: radial-gradient(ellipse at bottom, rgba(0, 212, 255, 0.15), transparent);
  pointer-events: none;
  z-index: -1;
}
```
